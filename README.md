# 🚚 Amazon Freight Logistics Efficiency Analysis

## 📌 Project Overview
This project focuses on analyzing Amazon freight logistics data to identify inefficiencies, delivery delays, and cost optimization opportunities. The goal is to simulate a real-world data analysis workflow using a combination of SQL and Python.

---

## 🎯 Objectives
- Analyze shipment delivery performance  
- Identify delay patterns and bottlenecks  
- Evaluate carrier efficiency  
- Optimize cost vs distance relationships  

---

## 🛠️ Tools & Technologies
- Python (Pandas, NumPy)
- SQL (Executed within Python)
- Matplotlib / Seaborn (Data Visualization)
- Jupyter Notebook

## 🧠 SQL + Python Integration
In this project, SQL queries were executed inside Python to perform data extraction and analysis.

### Key SQL Operations:
- SELECT, WHERE filtering  
- GROUP BY aggregations  
- JOIN operations  
- Subqueries for deeper insights  

This reflects real-world industry workflows where SQL and Python are used together for data analysis.
